To be run on two separate terminals


If executable permissions for chatserver.py are not in place then type

chmod +x chat_server.py

To run the server side:

python chat_server.py <server_port>


To compile chatclient.c: 

gcc chatclient.c -o chatclient

To run the client side: 

./chatclient <server_ip> <server_port>
